# Nova Inns Apartments — Frontend

Interfaz de gestión para apartamentos de alquiler corto. 100% dark mode.

## Tecnologías

- **Angular 21**
- **Tailwind CSS v4** — design system con paleta personalizada
- **TypeScript**

## Design System

| Token | Color | Uso |
|-------|-------|-----|
| `bg` | `#0B0B0C` | Fondo principal |
| `surface` | `#151518` | Cards / contenedores |
| `elevated` | `#1F1F23` | Superficies elevadas |
| `brand` | `#F2C200` | Botones, highlights, iconos activos |
| `brand-accent` | `#FFD21A` | Hover de brand |
| `brand-pressed` | `#C99A00` | Estado pressed |
| `border` | `#2A2A2E` | Bordes / divisores |
| `text-primary` | `#FFFFFF` | Texto principal |
| `text-secondary` | `#B3B3B8` | Texto secundario |

## Correr el proyecto

```bash
npm install
npm start
```
